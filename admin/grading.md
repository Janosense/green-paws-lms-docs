# Перевірка завдань

Сторінка **«Перевірка завдань»** — це черга робіт, зданих студентами й готових до оцінювання.

## Черга перевірки

1. У меню «Green Paws LMS» оберіть **«Перевірка завдань»**.
2. Ви побачите список робіт зі статусом **«очікує»**, відсортований від найстаріших.
3. Колонки: ім'я студента, назва завдання, курс, дата здачі.

## Оцінювання роботи

1. Натисніть на рядок, щоб відкрити роботу.
2. Угорі показано контекст: максимальний і прохідний бали, тип здачі, дедлайн, критерії оцінювання.
3. Нижче — здана робота: текст та/або прикріплений файл, ім'я та email студента.
4. У формі оцінювання:
   - Введіть **бал** (від 0 до максимального).
   - Додайте **відгук** для студента.
5. Натисніть **«Оцінити»**, щоб виставити бал, або **«Відхилити»**, щоб повернути роботу на доопрацювання.

## Після оцінювання

- Оцінена або відхилена робота **блокується** — повторно змінити оцінку не можна.
- Якщо бал не нижчий за прохідний, завдання зараховується, і система перераховує прогрес курсу студента.

::: warning Відгук при відхиленні
Для дії «Відхилити» бажано заповнити поле відгуку, щоб студент зрозумів, що саме виправити.
:::
