# Панель інструктора

Перша сторінка меню «Green Paws LMS» — оглядова панель з ключовими показниками ваших курсів.

## Що показує панель

Зверху розташовані картки з підсумками по всіх ваших курсах (тих, де ви автор або ко-інструктор):

- **Усього курсів** — скільки курсів ви ведете.
- **Усього записів** — скільки студентів записано на ваші курси разом.
- **Усього завершень** — скільки студентів повністю пройшли курси.

Нижче — таблиця по кожному курсу з колонками: назва, кількість записів, кількість завершень та кнопка **«Переглянути»**.

## Перегляд курсу очима інструктора

Кнопка **«Переглянути»** відкриває курс у режимі попереднього перегляду (`?preview=1`):

1. Ви потрапляєте у плеєр уроків, **не записуючись** на курс.
2. Обмеження доступу для незаписаних користувачів не діють — ви бачите весь матеріал.
3. Ваш прогрес **не зберігається** — перегляд не лишає слідів у статистиці.
4. Угорі показується банер, що ви переглядаєте курс як інструктор.

::: tip Порада
Використовуйте «Переглянути», щоб перевірити, як виглядає курс для студента, перш ніж відкривати запис.
:::
