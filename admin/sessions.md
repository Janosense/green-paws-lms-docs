# Когортні сесії

Сесія — це заняття наживо у Zoom для **когортного** курсу. Сесія належить до когортного курсу та автоматично синхронізується із Zoom.

## Створення сесії

1. У меню оберіть **«Сесії»** → **«Додати нову сесію»**.
2. Введіть **назву** та опис сесії.
3. Заповніть блок **«Параметри сесії»**:
   - **Курс** — оберіть когортний курс.
   - **Номер сесії** — порядковий номер у курсі.
   - **Початок** / **Завершення** — дата й час проведення.
   - **Статус** — `заплановано`, `наживо`, `завершено`, `скасовано`.
   - **Запис доступний до** — до якої дати студенти можуть переглядати запис.
   - **Матеріали** — додаткові файли для заняття.
4. Натисніть **«Опублікувати»**.

## Інтеграція Zoom

Після збереження сесії дані Zoom-зустрічі **підставляються автоматично** (потрібні налаштовані ключі Zoom — див. [Налаштування](/admin/settings)):

- **ID зустрічі**, **посилання для приєднання**, **посилання для старту** (для інструктора), **пароль**.
- **Посилання на запис** з'являється автоматично після завершення зустрічі.

Статус сесії також змінюється автоматично:

- `заплановано` → `наживо` — коли зустріч почалася;
- `наживо` → `завершено` — коли зустріч завершилася.

Інструктор може вручну встановити статус **«скасовано»**.

::: tip Порада
Якщо ключі Zoom ще не налаштовані, сесію все одно можна створити — посилання Zoom з'являться після під'єднання інтеграції.
:::
