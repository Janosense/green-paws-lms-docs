# Курси

Курс — це основна одиниця навчання. Усі інші елементи (модулі, уроки, теми, тести, завдання, сесії) належать до курсу.

## Створення курсу

1. У лівому меню оберіть **«Курси»** → **«Додати новий курс»**.
2. Введіть **назву курсу** та опис у головному редакторі.
3. Заповніть блок **«Параметри курсу»** (див. нижче).
4. За потреби додайте співавторів у блоці **«Ко-інструктори»** (див. [Ко-інструктори](/admin/co-instructors)).
5. Натисніть **«Опублікувати»**.

## Параметри курсу

### Загальне

- **Тип курсу** — оберіть один із двох:
  - **Самостійний (self-paced)** — студент проходить у власному темпі, без розкладу.
  - **Когортний (cohort)** — група студентів навчається за розкладом, із сесіями наживо.
- **Тривалість (годин)** — орієнтовний загальний час курсу.
- **Промо-відео** — посилання на відео для сторінки курсу.
- **Обкладинка** — зображення-мініатюра курсу.

### Ціна та доступ

- **Ціна** — у гривнях; `0` означає безкоштовний курс.
- **Валюта** — за замовчуванням UAH.
- **Відкрита реєстрація** — прапорець, що дозволяє запис на курс.
- **Максимум студентів** — `0` означає без обмежень.

### Когортні налаштування

Ці поля з'являються, лише якщо тип курсу — **когортний**:

- **Реєстрація відкривається** / **закривається** — вікно запису.
- **Курс починається** / **завершується** — дати проведення курсу.

### Завершення

- **Поріг проходження (%)** — який відсоток потрібен для зарахування прогресу.
- **Видавати сертифікат після завершення** — прапорець автоматичної видачі сертифіката.

::: tip Як зараховується завершення курсу
Курс вважається пройденим, коли студент завершив **усі уроки на 100%** *і* склав **фінальний іспит** (якщо він є). Якщо фінального іспиту в курсі немає — достатньо пройти всі уроки. Див. [Тести](/admin/quizzes).
:::

::: warning Самостійні vs когортні
Модулі додаються лише до **самостійних** курсів. Сесії наживо — лише до **когортних**. Оберіть тип уважно перед наповненням курсу.
:::

## Що далі

Після створення курсу наповніть його контентом:

- [Модулі](/admin/modules) — для групування уроків (самостійні курси).
- [Уроки](/admin/lessons) — основні навчальні одиниці.
- [Сесії](/admin/sessions) — заняття наживо (когортні курси).
