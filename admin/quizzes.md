# Тести та питання

Тест — це набір питань для перевірки знань. Тест можна прикріпити до курсу, модуля, уроку або сесії.

## Створення тесту

1. У меню оберіть **«Тести»** → **«Додати новий тест»**.
2. Введіть **назву** та опис тесту.
3. Заповніть блок **«Параметри тесту»**:
   - **Обмеження часу (секунд)** — `0` означає без обмеження.
   - **Максимум спроб** — `0` означає необмежено.
   - **Поріг проходження (%)** — за замовчуванням 80.
   - **Перемішувати питання** / **Перемішувати відповіді** — прапорці.
   - **Показувати правильні відповіді** — `ніколи`, `після відповіді` або `після проходження`.
   - **Це фінальний іспит** — прапорець (див. нижче).
   - **Батьківська одиниця** — курс/модуль/урок/сесія.
4. Натисніть **«Опублікувати»**.
5. Додайте питання у блоці зі списком питань кнопкою **«Додати запитання»**.

## Додавання питань

1. На сторінці тесту натисніть **«Додати запитання»** (батьківський тест підставиться автоматично).
2. Заповніть блок **«Питання та відповіді»**:
   - **Тип питання** — `одна відповідь`, `кілька відповідей`, `так/ні` або `текстова відповідь`.
   - **Бали** — скільки балів дає питання.
   - **Медіа** — необов'язкове зображення.
   - **Пояснення** — показується студенту після відповіді.
3. Для **текстових** питань: вкажіть **режим звірки** (точно / без урахування регістру / regex) та **правильну відповідь**.
4. Для питань із вибором: у конструкторі відповідей додайте варіанти, позначаючи правильні.
5. Натисніть **«Опублікувати»**.

Питання можна перетягувати, щоб змінити порядок — див. [Порядок навчального плану](/admin/curriculum-order).

## Фінальний іспит

Прапорець **«Це фінальний іспит»** робить тест умовою завершення курсу:

::: tip Як це працює
Якщо в курсі є тест із позначкою «фінальний іспит», курс зараховується студенту лише після того, як він **пройде всі уроки на 100%** *і* **складе фінальний іспит**. Якщо фінального іспиту немає — достатньо пройти всі уроки.
:::
