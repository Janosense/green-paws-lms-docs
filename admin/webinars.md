# Вебінари

Вебінар — це окрема онлайн-подія у Zoom, не пов'язана з курсами. Студенти реєструються на вебінар окремо.

## Створення вебінару

1. У меню оберіть **«Вебінари»** → **«Додати новий вебінар»**.
2. Введіть **назву**, опис, додайте обкладинку та промо-відео.
3. Заповніть блок **«Параметри вебінару»** (див. нижче).
4. Натисніть **«Опублікувати»**.

## Параметри вебінару

### Розклад

- **Початок** / **Завершення** — дата й час проведення.
- **Статус** — `заплановано`, `наживо`, `завершено`, `скасовано` (змінюється автоматично через Zoom).

### Реєстрація та ціна

- **Ціна** — у гривнях; `0` означає безкоштовно.
- **Валюта** — за замовчуванням UAH.
- **Максимум учасників** — `0` означає без обмежень.
- **Реєстрація відкривається** / **закривається** — вікно реєстрації.

### Запис

- **Днів доступу до запису** — скільки днів запис доступний після вебінару; `0` означає, що запису не буде.
- **Посилання на запис** з'являється автоматично після завершення.

### Промо

- **Промо-відео** та **обкладинка** для сторінки вебінару.

## Інтеграція Zoom

Як і сесії, вебінари автоматично синхронізуються із Zoom (ID зустрічі, посилання, пароль). Потрібні налаштовані ключі Zoom — див. [Налаштування](/admin/settings).

::: tip Відмінність від сесій
Сесія — це заняття всередині когортного курсу. Вебінар — самостійна подія в публічному каталозі, на яку реєструються окремо.
:::
