# Модулі

Модуль — це група уроків усередині курсу. Модулі допомагають структурувати великі курси на логічні розділи. Використовуються лише в **самостійних** курсах.

## Створення модуля

1. У меню оберіть **«Модулі»** → **«Додати новий модуль»**.
2. Введіть **назву модуля** та за потреби опис.
3. Заповніть блок **«Параметри модуля»**:
   - **Курс** — оберіть батьківський курс зі списку (показуються лише самостійні курси). Можна лишити «Без курсу».
   - **Вступне відео** — необов'язкове посилання.
   - **Тривалість (хвилин)** — для відображення.
   - **Поріг проходження (%)** — для розрахунку прогресу модуля.
4. Натисніть **«Опублікувати»**.

## Список модулів

У списку модулів є колонки:

- **Курс** — до якого курсу належить модуль.
- **Уроки** — скільки уроків містить модуль.

Над списком є фільтр за курсом — оберіть курс, щоб побачити лише його модулі.

## Порядок модулів

Модулі в межах курсу можна **перетягувати**, щоб змінити їхній порядок. Див. [Порядок навчального плану](/admin/curriculum-order).

::: tip Порада
Модулі необов'язкові. Якщо курс невеликий, уроки можна прикріпити безпосередньо до курсу без модулів — див. [Уроки](/admin/lessons).
:::
