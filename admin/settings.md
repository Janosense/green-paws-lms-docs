# Налаштування

Сторінка **«Налаштування»** містить ключі інтеграцій: Zoom (заняття наживо) та LiqPay (оплати). Доступна адміністраторам.

<LoomEmbed id="" title="Налаштування Zoom та LiqPay" />

## Налаштування Zoom

Потрібні для автоматичного створення Zoom-зустрічей у сесіях і вебінарах.

1. У меню «Green Paws LMS» оберіть **«Налаштування»**.
2. У розділі **Zoom** заповніть:
   - **Account ID**;
   - **Client ID**;
   - **Client Secret**.
3. Натисніть **«Зберегти зміни»**.

Ключі беруться з кабінету Zoom (Server-to-Server OAuth застосунок).

## Налаштування LiqPay

Потрібні для прийому оплат за курси та вебінари.

1. У розділі **LiqPay** заповніть:
   - **Merchant ID**;
   - **Публічний ключ**;
   - **Приватний ключ**;
   - **Тестовий режим (sandbox)** — прапорець для перевірок без реальних платежів.
2. Натисніть **«Зберегти зміни»**.

Після збереження вгорі з'явиться повідомлення **«Налаштування збережено»**.

::: warning Безпека ключів
Приватний ключ LiqPay і Client Secret Zoom — конфіденційні. Не передавайте їх стороннім. Якщо ключі вже задані на рівні сервера (константами), форма їх не перезаписує.
:::
