# Порядок навчального плану

Порядок елементів у курсі задається простим перетягуванням мишею (drag-and-drop).

<LoomEmbed id="" title="Зміна порядку елементів курсу" />

## Що можна впорядковувати

Перетягуванням змінюється порядок:

- **Модулів** усередині курсу;
- **Уроків** усередині модуля (або курсу);
- **Тем** усередині уроку;
- **Питань** усередині тесту.

## Як змінити порядок

1. Відкрийте на редагування батьківський елемент (наприклад, курс — щоб упорядкувати модулі, або урок — щоб упорядкувати теми).
2. Знайдіть блок зі списком дочірніх елементів.
3. Наведіть курсор на елемент, затисніть ліву кнопку миші та **перетягніть** його на нову позицію.
4. Відпустіть — новий порядок **зберігається автоматично**, окремо натискати «Зберегти» не потрібно.

::: tip Порада
Якщо щось не перетягується — оновіть сторінку. Зміни, внесені іншим редактором, можуть оновитися лише після перезавантаження.
:::

::: warning Сесії та завдання
Когортні **сесії** не впорядковуються перетягуванням — вони шикуються за датою проведення. **Завдання** також не мають ручного порядку.
:::
