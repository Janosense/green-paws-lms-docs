# Сертифікати

За успішне проходження курсу ви можете отримати сертифікат (якщо курс його передбачає).

<LoomEmbed id="" title="Отримання сертифіката" />

## Як отримати сертифікат

1. Повністю пройдіть курс: усі уроки та фінальний іспит (якщо він є).
2. Сертифікат видається **автоматично** після завершення.
3. Знайдіть його у своїй особистій панелі в розділі сертифікатів.
4. Сертифікат можна **завантажити у форматі PDF**.

::: tip Не всі курси видають сертифікат
Сертифікат доступний лише для курсів, де інструктор увімкнув його видачу. Якщо після завершення сертифіката немає — цей курс його не передбачає.
:::

::: warning Повернення коштів
Якщо за курс оформлено повернення коштів, виданий сертифікат анулюється.
:::
